# CthulhuCommunicationTool

- クトゥルフ用のツールです。

## 使い方
1. node.jsをインストールします
1. コンソールでダウンロードしてきたフォルダに移動する
1. "node server.js"とコマンドを打つ
1. "serval running"と出てきたら起動完了

## 機能
- ツールバーメニュー
	- ダイス(チャット欄に自動共有されます)
	- 情報テーブル(必要な情報(と勝手に思っている)だけ表示するようにしてます。)
	- 戦闘送信(戦闘時に必要な情報をチャット欄に流します。)
	- 画像送信(表示する名前とURLを入れると、画像共有の方にボタンが増えます。)
- タブ
	- チャット(テキストボックスに入れて送信を押すと送信)
	- 画像共有(お絵かきできて、さらにその情報が共有できます。)


### 線の送信ルール
####必要な情報
- 開始座標
- 終了座標
- 色
- 太さ
- 消しゴム
- 全消し
